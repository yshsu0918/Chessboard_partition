# Chessboard_partition
切棋譜會需要用到的功能

## 功能們

### 切棋譜
1. 多個題目在一譜中 => 一題一譜
2. 一題一譜 => spined_sgf (旋轉到右上角)
3. spined_sgf => 套用mask => maskedsgf
4. 多個 maskedsgf => 生成 CGI 測試腳本
5. 多個 maskedsgf => 生成 ELF 測試腳本
